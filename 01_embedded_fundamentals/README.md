# Embedded Fundamentals

This folder contains small Arduino and AVR embedded-programming examples.  
The examples demonstrate basic microcontroller concepts before moving to larger and more integrated C/C++ projects.

## Examples

| File | Technical concept |
|---|---|
| `Digital_Writing_Code.ino` | Register-level GPIO output control using AVR registers |
| `Led_Brightness_By_PWM.ino` | LED brightness control using PWM |
| `Controlling_Servo_Motor.ino` | Basic servo-motor position control |
| `Interrupt_Service_Routine.ino` | Basic input and event-handling example |
| `Seven_Segment_Display.ino` | Seven-segment display control using digital outputs |

## Hardware Platform

The examples are primarily designed for:

- Arduino Uno
- ATmega328P microcontroller
- Arduino IDE
- Embedded C/C++

## Topics Covered

- Digital input and output
- AVR register-level programming
- PWM generation
- Servo-motor control
- Event handling
- Seven-segment display interfacing
- Basic embedded C/C++ programming

## Project Level

These files are foundational examples. More advanced projects involving sensors, communication protocols, state machines, fault handling, and automation control are maintained separately in this repository.
